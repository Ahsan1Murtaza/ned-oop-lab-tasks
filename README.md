# 🧩 C++ OOP Lab Exercises

This repository contains a structured set of C++ lab tasks focused on learning and applying Object-Oriented Programming (OOP) concepts.

It includes assignments and problem-solving exercises designed to strengthen understanding of class design and object-oriented principles.

---

## 📚 Topics Covered

- Classes and Objects
- Constructors and Destructors
- Encapsulation
- Inheritance
- Polymorphism
- Function Overloading
- Operator Overloading
